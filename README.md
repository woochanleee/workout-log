## 개요

운동을 하고 글, 사진을 올리며 기록을 남기는 블로그형 사이트

## 의의

- 운동을 하고 글을 올리는 애플리케이션이 다양하지 않음, 예외) 운동스타그램
- 다른 사람들의 운동 기록을 보며 동기부여를 받을 수 있음
- 운동에 관련된 글을 중심으로 볼수 있음
- 글을 올림으로써 운동을 몇일 째 하는 중이고, 몇일 쉬었는지 확인할 수 있어 스스로 자극을 받음
